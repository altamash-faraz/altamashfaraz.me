## 🔧 Local Development

1. Clone the repository
2. Open `index.html` in a web browser
3. For development, use a local server:

   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve .

   # Using PHP
   php -S localhost:8000
   ```
